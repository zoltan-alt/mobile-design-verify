# Pet-care home — moodboard summary

User shared a single small moodboard image showing the top portion of a
pet-care app home screen. Aesthetic: warm cream background, handwritten
greeting, decorative stickers, organic mood-board feel. App is for
zoltan-alt's `pet-ops` (offline pet care record keeper).

## Source references

- Moodboard image attached in user message (extracted from session JSONL
  via PowerShell `ConvertFrom-Json` + `FromBase64String`, see
  `skill_extract_attached_images.md`).
- Pet info: Milo / Tabby Cat.

## Vibes

- Hand-drawn / hand-painted feel
- Warm cream + soft purple as the primary palette
- Coral pink as accent
- Slight tilts and asymmetry — no rigid grid
- Section headers are casual handwritten Caveat with hand-drawn
  highlighter behind

## Constraints

- Image placeholders are diagonal-dashed (per project rule — no
  generated photos).
- Icons in tab bar from a 5-PNG asset set (home, calendar, cat, dog,
  heart) the user provided. Tab-bar only — no reuse of these icons
  elsewhere in the design.
